# 04-Records

## 역할
이 섹션은 프로젝트의 작업, 문제, 결정 이력을 관리한다.  
`03-Status`가 현재와 다음을 설명한다면, 이 섹션은 언제 무엇을 했고 왜 그렇게 판단했는지를 남긴다.

## 구성
- [[wiki/04-Records/Worklog/README|Worklog]] - 작업 수행 이력
- [[wiki/04-Records/Issues/README|Issues]] - 문제 분석과 블로커 기록
- [[wiki/04-Records/Decisions/README|Decisions]] - 설계/기술 선택 근거

## 사용 규칙
- 현재 상태 요약은 [[wiki/03-Status/Current-State|Current State]]에 반영한다.
- 다음 작업 우선순위는 [[wiki/03-Status/Next-Work|Next Work]]에 반영한다.
- 원본 사실은 [[wiki/05-Sources/README|05-Sources]]로 링크한다.
