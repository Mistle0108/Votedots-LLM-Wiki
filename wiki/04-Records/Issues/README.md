# Issues

## 역할
이 폴더는 프로젝트 진행 중 드러난 구조적 문제를 분석하고 기록한다.  
원본 issue 사실이 아니라, 증상과 원인, 대응, 남은 리스크를 정리하는 계층이다.

## 연결 문서
- 기록 메인: [[wiki/04-Records/README|04-Records]]
- 작업 이력: [[wiki/04-Records/Worklog/README|Worklog]]
- 결정 이력: [[wiki/04-Records/Decisions/README|Decisions]]
- issue source: [[wiki/05-Sources/issues/README|Issue Sources]]

## 구분 기준
- `wiki/05-Sources/issues/`는 GitHub issue 원문 사실 요약 계층이다.
- 이 폴더는 코드와 작업 맥락을 기반으로 한 문제 분석 계층이다.
