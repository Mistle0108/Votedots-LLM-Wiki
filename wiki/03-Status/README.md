# 03-Status

## 역할
이 섹션은 현재 상태와 다음 작업을 관리하는 공식 문서다.

## 문서 구성
- [[wiki/03-Status/Current-State|Current State]] - 지금 어디까지 구현됐는지
- [[wiki/03-Status/Next-Work|Next Work]] - 다음에 무엇을 해야 하는지

## 읽는 기준
- 지금 상태를 알고 싶으면 `Current-State`를 먼저 본다.
- 바로 이어서 해야 할 작업을 알고 싶으면 `Next-Work`를 본다.
- 왜 그런 상태가 되었는지는 `04-Records`를 본다.
