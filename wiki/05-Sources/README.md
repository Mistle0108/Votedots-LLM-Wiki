# 05-Sources

## 역할
이 섹션은 raw 원본을 사실 중심으로 요약한 source 계층이다.

## 구성
- [[wiki/05-Sources/repos/README|Repo Sources]] - repo overview와 module-level source 색인
- [[wiki/05-Sources/issues/README|Issue Sources]] - open issue 사실 요약
- [[wiki/05-Sources/prs/README|PR Sources]] - 핵심 merged PR 사실 요약

## 규칙
- 이 섹션에는 사실만 적는다.
- 해석은 `01-Project`, `02-Architecture`, `03-Status`, `04-Records`에 적는다.
- 새 source가 생기면 상태/기록 문서에 반영이 필요한지 같이 판단한다.
